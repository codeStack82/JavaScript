JavaScript

JavaScript is high-level Programming Language that is commonly used in conjunction with HTML and CSS. JavaScript adds additional abilities, interaction and complexity to web pages JavaScript is becoming very popular in the last few years and many developers are using to create websites more advanced websites than were previously possible without it. JavaScript is supported on all current internet browsers. Big name companies like Google and Facebook are using and even creating their own flavors of JavaScript.



Document Object Model

When a web page is rendered in the browser an object model is created. This object model is known as the Document Object Model or simply as the DOM. The DOM is a tree or heirarchy structure that represents each element in a web page. The DOM allows other programming languages like CSS and JavaScript to interact with the elements or objects within a webpage.



AJAX

Ajax or Asynchronous JavaScript and XML allows webpages to easily communicate with the web-server via custom server-side scripts. One of the main advantages that programmers have when using Ajax is its Asynchronous functionality. Essentially, this allows the Ajax scripts to run, update and communicate with the server without having to refresh the entire webpage.



JQuery

JQuery is a JavaScript library that simplifies and streamlines Html document manipulation. JQuery was released ten years ago and is now one of the most widely used JavaScript libraries in use. jQuery essentially simplifies using the DOM making it easier to create dynamic web pages.
